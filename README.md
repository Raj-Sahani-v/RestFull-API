# User Management Admin Panel (REST API)
A RESTful API based User Management Admin Panel built using Node.js, Express, and MySQL. 
The system allows administrators to perform full CRUD operations and manage users efficiently.
## Tech Stack
- Node.js
- Express.js
- MySQL
- EJS
- HTML, CSS
- RESTful API
## Features
- View all users
- Add new user
- Update user
- Delete user
- Database integration
